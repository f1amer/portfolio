# IT Support Portfolio (Bootstrap + jQuery)

A modern, Netflix-inspired single-page portfolio built with:
- HTML + CSS
- Bootstrap 5
- jQuery
- AOS (scroll animations)

## Run locally (no install)
Just open `index.html` in your browser.

## Run with a local server (recommended)
### Option A: VS Code Live Server
Install the **Live Server** extension → right-click `index.html` → **Open with Live Server**

### Option B: Python server
```bash
cd it-support-portfolio
python -m http.server 5500
```
Then open: http://localhost:5500

## Customize
- Edit your name, links, and content in `index.html`
- Edit colors, spacing, and theme in `assets/css/styles.css`
- Edit interactions (filters, theme toggle, contact mailto) in `assets/js/app.js`

## Add your resume
Put your resume file inside:
`assets/docs/`
Then update the download button link in `index.html`.
